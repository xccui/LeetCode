Given _n_, generate all structurally unique **BST's** (binary search trees) that store values 1..._n_ .

For example,

Given _n_ = 3, your program should return all 5 unique BST's shown below.

    1         3     3      2      1
     \       /     /      / \      \
      3     2     1      1   3      2
     /     /       \                 \
    2     1         2                 3
